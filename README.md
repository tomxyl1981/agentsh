# AgentSH 电子杂志网站

## 项目概述
这是一个体现九紫离火运主题的AI新闻电子杂志网站，采用高贵紫色和盛唐红色调设计，为每篇新闻提供独立的精美页面。

## 设计特色
- **九紫离火运主题**：高贵紫色(#9932CC) + 盛唐红色(#8B0000)
- **电子杂志质感**：现代化排版，丰富交互动效
- **响应式设计**：适配各种设备屏幕
- **完整内容**：每篇文章都有详实的内容而非空页面

## 技术特点
- **纯静态网站**：HTML5 + CSS3 + JavaScript
- **现代化UI**：渐变背景、卡片设计、流畅动画
- **SEO友好**：语义化HTML，完整meta标签
- **性能优化**：轻量级代码，快速加载

## 部署说明
1. 将整个目录推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为源
4. 网站将自动部署到 `username.github.io/repository-name`

## 开发建议
- 使用本地服务器进行开发测试
- 定期优化性能和用户体验
- 保持代码简洁和可维护性
